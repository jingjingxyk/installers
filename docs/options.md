# 配置选项

## --latest

    每次执行都执行 git克隆源码

## --install-php

    系统未安装 PHP , 调用系统包管理工具 安装 PHP ，已安装 PHP 将不会执行安装 PHP

## --swoole-debug

    swoole 开启 debug 编译 参数

## --swoole-version

     指定swoole 版本

## --mirror

    指定 swoole 源码地址
    选项值： china

## --swoole-test

    swoole 编译完成以后，执行swoole测试代码

## --debug

    显示shell 正在执行的命令

## --install-phpy

    安装 phpy 扩展

## --phpy-vesion

    指定 phpy 扩展版本

## --install-python3

    系统未安装 python3 , 调用系统包管理工具 安装 python3 ，已安装 python3 将不会执行安装 python3
